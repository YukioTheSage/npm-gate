# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experience levels.
- Accept constructive feedback.
- Keep security discussions responsible and defensive.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private vulnerability details before coordinated disclosure.
- Submitting malware, credential theft, exploit payloads, or harmful examples.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or ban participants who violate this code.
